# **SDG 3 Indicator Text Classification**

Multi‑label text classification system that predicts which Sustainable Development Goal 3 (SDG 3) indicators are relevant to a given document (grants, tenders, reports, news, etc.).  
Developed as a group assignment using classical machine learning (TF‑IDF, Logistic Regression, Random Forest, Linear SVM, Word2Vec) with 8 systematic experiments.

## **How to Run on Google Colab**

1. **Open the notebook** in Colab:  
   [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/YOUR_USERNAME/SDG_Indicator_Text_Classification/blob/main/notebooks/02_MemberB_Modeling_Experiments.ipynb)  
   (Replace `YOUR_USERNAME` with your GitHub username and adjust the notebook path as needed)

2. **Mount Google Drive** when prompted.

3. **Set up the shared Drive folder** – make sure the folder `/content/drive/MyDrive/SDG/` exists and contains:
   - `Devex_train.csv`
   - `Devex_test_questions.csv`
   - `train_processed.csv` (generated by Member A)

4. **Run all cells** in order (Runtime → Run all).  
   The notebook will:
   - Load preprocessed data or raw CSVs
   - Create TF‑IDF features
   - Train 8 different models/experiments
   - Save the best model (`best_model_svm.pkl`) and results (`all_experiment_results.csv`) to the Drive folder

5. **For end‑to‑end reproduction**, start from Member A’s notebook (`01_MemberA_Data_Pipeline.ipynb`) to generate the preprocessed data, then continue with Member B, C, and D notebooks.
